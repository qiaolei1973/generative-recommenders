## 🎯 学习路径

### 快速上手（1-3天）
```
第1天：01 核心算法详解 → 理解 HSTU、STU、Sampled Softmax
第2天：02 数据集说明 → 了解数据格式和含义
第3天：03 快速参考手册 → 熟悉常用命令和参数
```

### 深入理解（1-2周）
```
第1周：04 传统vs生成式推荐 → 理解范式转变
第2周：08 评估体系详解 → 掌握评估方法
```

### 实践建议
```
1. 先通读 01-核心算法详解，建立整体认知
2. 边看 03-快速参考手册，边动手实践
3. 遇到问题查阅对应的专题文档
4. 定期回顾快速参考手册巩固知识
```
